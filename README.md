# Flask Task Manager

A simple task management web application built with Flask and SQLite.

## Features

- ✅ Create, Read, Update, Delete tasks
- ✅ Mark tasks as complete/pending
- ✅ Search tasks
- ✅ Due dates with overdue highlighting
- ✅ Responsive design
